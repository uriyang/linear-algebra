## 3 성질이 나쁜 경우
### 3.1 성질이 나쁜 예
- 단서가 부족한 경우(가로가 긴 행렬, 핵)
  - y의 차원이 x보다 작은 경우 (m < n)
  - 여러 입력 x가 같은 출력 y로 눌려 들어간다.
  - 이런 경우 일부 정보가 구별되지 않고 사라진다.
  - A의 핵(kernel)
    - Ax = o가 되는 x의 집합
    - 핵이 원점 말고도 크면, 서로 다른 입력이 같은 결과로 간다.
- 단서가 너무 많은 경우(세로가 긴 행렬, 상)
  - y의 차원이 x보다 큰 경우 (m > n)
  - 만들 수 있는 출력의 종류가 제한될 수 있다.
  - 모든 y를 다 만들지 못하면 전사가 아니다.
- 단서의 개수가 일치해도 성질이 나쁜 경우(특이행렬)
  - 정방행렬이라도 납작하게 눌리면 역행렬이 없다.

### 3.2 성질이 나쁜 경우와 핵・상
- 핵(Ker A)
  - A에 의해 원점으로 보내지는 입력의 집합
  - 핵이 클수록 사라지는 정보가 많다.
- 상(Im A)
  - A로 실제 만들어낼 수 있는 출력의 집합
  - 상이 출력공간 전체보다 작으면, 도달할 수 없는 y가 있다.
- 단사
  - 같은 결과 y가 나오는 원인 x는 유일한가
  - Ker A = {o} 이면 단사
- 전사
  - 어떤 결과 y에도 그것이 나오는 원인 x가 존재하는가
  - Im A가 출력공간 전체이면 전사
- 전단사
  - 단사와 전사가 동시에 성립하는 경우
  - 이때 역행렬이 존재한다.

### 3.3 차원 정리
- 차원 정리: 전체 차원 = 살아남은 차원 + 사라진 차원
- 수식
  - dim(입력공간) = rank(A) + nullity(A)
- 해석
  - rank(A): 출력으로 살아남은 독립적인 방향 수
  - nullity(A): 핵의 차원, 즉 사라진 방향 수

### 3.4 '납작하게'를 식으로 나타내다(선형독립, 선형종속)
- 선형종속
  - a<sub>1</sub>x<sub>1</sub> + ... + a<sub>n</sub>x<sub>n</sub> = 0 이 되는 해가 있을 때,
  - 적어도 하나의 계수가 0이 아닌 해가 존재
  - 어떤 벡터가 다른 벡터들의 조합으로 표현된다.
  - 납작하게 눌릴 가능성이 생긴다.
- 선형독립
  - a<sub>1</sub>x<sub>1</sub> + ... + a<sub>n</sub>x<sub>n</sub> = 0 의 해가
  - 모든 계수가 0인 경우뿐
  - 서로 중복 없이 독립적인 방향을 이룬다.
- 기저와 선형독립
  - 기저는 선형독립이면서 공간 전체를 표현할 수 있는 벡터 집합

### 3.5 단서의 실질적인 개수(랭크)
- 랭크의 정의
  - 몇 개의 독립적인 방향이 살아남았는가
  - 상(Im A)의 차원
- 랭크와 핵, 상과 단사, 전사
  - rank(A) = n 이면 단사
    - 입력 정보가 하나도 안 사라진다.
  - rank(A) = m 이면 전사
    - 출력공간을 전부 커버한다.
  - 정방행렬에서는 rank(A) = n = m 이면 단사, 전사, 역행렬 존재가 모두 같다.
- 랭크의 기본 성질
  - rank(A) ≤ m
  - rank(A) ≤ n
  - 랭크는 행 수나 열 수보다 클 수 없다.
- 보틀넥 형의 분해
  - 랭크가 r인 행렬 A는
  - A = BC 로 분해할 수 있다.
  - 이때 B는 m x r, C는 r x n 형태로 볼 수 있다.
  - 즉 A의 핵심 정보는 r차원 병목을 거쳐 전달된다.
- 전치해도 랭크는 동일
  - rank(A) = rank(A<sup>T</sup>)
  - 행 기준으로 봐도, 열 기준으로 봐도 살아남은 독립 방향 수는 같다.

## 4 성질이 좋고 나쁨의 판정 (역행렬)
### 4.1 '납작하게 눌리는가'가 포인트
- '납작하게 눌리지 않는다'는 'Ker A가 원점 o뿐'이라고 말할 수 있다.
- 정방행렬에서는 이것이 rank(A) = n 과 같다.

### 4.2 정칙성과 같은 조건 여러 가지
- A가 정칙이다.
- det(A) != 0
- Ker A = {o}
- rank(A) = n
- Ax = y 가 모든 y에 대해 유일한 해를 갖는다.
- 열벡터들이 선형독립이다.
- 행벡터들이 선형독립이다.

### 4.3 정칙성의 정리
- 정방행렬에서는 위 조건들이 서로 모두 동치이다.
- 즉 하나만 확인해도 나머지를 함께 판정할 수 있다.

## 5 성질이 나쁜 경우의 대책
### 5.1 구할 수 있는 데까지 구한다 (1) 이론편
- 해가 존재하는가?
  - y가 Im A에 속해야 한다.
  - Im A: A에 따라 실제로 움직일 수 있는 출력의 집합
- 해를 모두 찾자
  - 특해: Ax = y 를 만족하는 해 하나 x<sub>0</sub>
  - 동차방정식 Ax = 0 의 일반해를 구한다.
  - Ax = y 의 모든 해는
  - 특해 + 동차방정식의 일반해

### 5.2 구할 수 있는 데까지 구한다 (2) 실전편
- 단서가 너무 많은 전형적인 예(해 없음)
  - 계산 끝에 0 = 4 같은 식이 나오면 모순이므로 해가 없다.
- 단서가 부족한 전형적인 예(해가 많음)
  - 자유변수에 임의의 값을 넣어 여러 해를 만든다.
- 도중에 길이 막힌 경우
  - 행이나 열의 순서를 바꾸어 피벗을 찾는다.
- 도중에 정말로 벽에 부딪힌 경우
  - 모순이 확인되면 해가 없다.
- 정리
  - 마지막 행이 모두 0이면 그 줄은 새로운 제약을 주지 않는다.
  - 마지막 행이 0 = c (c != 0) 꼴이면 해가 없다.

### 5.3 최소제곱법
- 해가 정확히 없더라도, 오차가 가장 작은 해를 구할 수 있다.
- 목표
  - ||Ax - y||^2 를 최소화하는 x를 찾는다.
- 의미
  - y를 정확히 맞추지는 못해도, 가장 가까운 출력 Ax를 만든다.
- AI와의 연결
  - 데이터에 노이즈가 있거나 식이 딱 맞지 않을 때
  - 손실 함수를 최소화하는 방식으로 학습하는 것과 연결된다.
- 특이값분해(SVD)나 일반화역행렬(pseudoinverse)이 이때 유용하다.
