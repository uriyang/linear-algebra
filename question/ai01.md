# AI 예제 1

다음 두 벡터가 주어져 있다.
```
v1 = (1,0)
v2 = (0,1)
```
이 벡터들의 선형결합으로
```
2v1 + 3v2
```
를 계산하라.

또한 이것이 좌표 공간에서 어떤 의미를 가지는지 설명하라.

---
(2,0) + (0,3) = (2, 3)
x 방향으로 2
y 방향으로 3
